# Oracle Knowledgebase

This repository contains a static Oracle Database knowledgebase web page.

## Structure

```text
.
├── index.html
├── reports/
│   └── repo-report.html
└── README.md
```

## Files

- `index.html` - Main Oracle knowledgebase page. Keep this file at the repository root so GitHub Pages can serve it directly.
- `reports/repo-report.html` - Local repository summary report.
- `README.md` - Repository overview and structure.

## Published Page

If GitHub Pages is enabled for the `main` branch from the repository root, the page is available at:

```text
https://pp-nextgen-dba.github.io/oracle/
```
